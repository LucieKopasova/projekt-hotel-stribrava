# Hotel Stříbrava
Webová aplikace pro správu rezervací hotelu Stříbrava.  
Uživatel může prohlížet dostupné pokoje a vytvářet nové rezervace.

##  Funkce
- Přehled pokojů  
- Rezervace pokojů  
- Správa dat přes API

## 🛠️Technologie
- React
- JavaScript  
- REST API (pokud nějaké máš)
